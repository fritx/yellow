# yellow

> A fork of the slides plugin on <http://www.creativision.cn>

Demo: <http://fritx.github.io/yellow/demo>

## Usage

```jade
#my_slides.slides
  ul.slide-pic
    li
      h3 Picture 1
      img(src="1.png")
    li
      h3 Picture 2
      img(src="2.png")
    li
      h3 Picture 3
      img(src="3.png")
```

```js
$('#my_slides').yellow()
```
